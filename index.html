dm-groonga-adapter by hryk

Datamapper search adapter for groonga search engine

dm-groonga-adapter provides is-search adapter for groonga (http://groonga.org). With dm-is-search, you can use groonga as search repository. Currently, This module supports groonga 0.1.7 or later. This is Alpha release. If you find bugs, please report by github issue.

Dependencies

  • groonga (ruby binding) >= 0.9.1
  • dm-core >= 0.10.0
  • dm-more >= 0.10.0

Install

    git clone git://github.com/hryk/dm-groonga-adapter.git

    cd dm-groonga-adapter

    rake install
